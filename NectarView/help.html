Getting Started

はじめに

NectarView is a high-speed comic viewer for macOS. To start using NectarView:

NectarViewはmacOS用の高速漫画ビューアです。使用を開始するには：

  1. Drag and drop image files, PDF files, or archives (ZIP, RAR, 7z) onto the application window.画像ファイル、PDFファイル、またはアーカイブ（ZIP、RAR、7z）をアプリケーションウィンドウにドラッグ&ドロップします。
  2. Alternatively, use the 'Open' option from the File menu or press Command+O.または、ファイルメニューの「開く」を使用するか、Command+Oを押します。
  3. NectarView will automatically display the images.NectarViewが自動的に画像を表示します。

Navigation

ナビゲーション

  • Use left and right arrow keys to switch between images.
  • 左右の矢印キーで画像を切り替えます。
  • Use the mouse wheel or trackpad to scroll vertically.
  • マウスホイールまたはトラックパッドで垂直スクロールします。
  • Double-click to toggle fullscreen mode.
  • ダブルクリックでフルスクリーンモードを切り替えます。

View Modes

表示モード

NectarView offers three view modes:

NectarViewは3つの表示モードを提供します：

  • Single Page View (Command+1): One page at a time.
  • 単ページビュー（Command+1）：1ページずつ表示。
  • Spread View R→L (Command+2): Two pages, right page first.
  • 見開きビュー 右→左（Command+2）：右ページから表示。
  • Spread View L→R (Command+3): Two pages, left page first.
  • 見開きビュー 左→右（Command+3）：左ページから表示。

Auto Page Turn

自動ページめくり

Enable auto page turn in the settings menu. Adjust the interval from 0.5 to 30 seconds.

設定メニューで自動ページめくりを有効にします。間隔は0.5〜30秒で調整可能です。

カスタマイズ

設定（Command+,）でカスタマイズ：

  • 背景色
  • Control bar color
  • コントロールバーの色
  • Reading direction
  • 読む方向

  • 言語設定

  • 画像：PNG、JPEG、GIF、BMP、TIFF、WebP

  • アーカイブ：ZIP、RAR、7z、TAR、GZ、BZ2、XZ、LHA/LZH、CAB、SIT/SITX
