Getting Started

はじめに

NectarView is a high-speed comic viewer for macOS. To start using NectarView:

NectarViewはmacOS用の高速漫画ビューアです。NectarViewの使用を開始するには：

  1. Drag and drop image files, PDF files, or ZIP archives onto the application window.
  2. 画像ファイル、PDFファイル、またはZIPアーカイブをアプリケーションウィンドウにドラッグ＆ドロップします。
  3. Alternatively, use the 'Open' option from the File menu or press Command+O to select files.
  4. または、ファイルメニューの「開く」オプションを使用するか、Command+Oを押してファイルを選択します。
  5. NectarView will automatically display the images in the selected file or folder.
  6. NectarViewは選択されたファイルまたはフォルダ内の画像を自動的に表示します。

Navigation

ナビゲーション

  • Use left and right arrow keys to switch between images.
  • 左右の矢印キーを使用して画像を切り替えます。
  • Use the mouse wheel or trackpad to scroll vertically within an image.
  • マウスホイールまたはトラックパッドを使用して画像内を垂直方向にスクロールします。
  • Press spacebar to toggle fullscreen mode.
  • スペースバーを押してフルスクリーンモードを切り替えます。
  • Double-click to toggle window size between fit-to-screen and actual size.
  • ダブルクリックでウィンドウサイズを画面に合わせるサイズと実際のサイズで切り替えます。

View Modes

表示モード

NectarView offers three view modes:

NectarViewは3つの表示モードを提供します：

  • Single Page View (Command+1): Displays one page at a time.
  • 単ページビュー（Command+1）：1ページずつ表示します。
  • Spread View (Right to Left) (Command+2): Displays two pages side by side, right page first.
  • 見開きビュー（右から左）（Command+2）：2ページを並べて表示し、右ページから始まります。
  • Spread View (Left to Right) (Command+3): Displays two pages side by side, left page first.
  • 見開きビュー（左から右）（Command+3）：2ページを並べて表示し、左ページから始まります。

Bookmarks

ブックマーク

NectarView allows you to bookmark your favorite pages:

NectarViewでは、お気に入りのページをブックマークできます：

  • Add/Remove Bookmark: Command+B
  • ブックマークの追加/削除：Command+B
  • Go to Next Bookmark: Command+]
  • 次のブックマークへ：Command+]
  • Go to Previous Bookmark: Command+[
  • 前のブックマークへ：Command+[
  • Show Bookmark List: Command+E
  • ブックマークリストの表示：Command+E

Auto Page Turn

自動ページめくり

Enable auto page turn in the settings menu. You can adjust the interval to set how quickly pages turn automatically.

設定メニューで自動ページめくりを有効にします。間隔を調整して、ページが自動的にめくられる速さを設定できます。

カスタマイズ

設定メニュー（Command+,）にアクセスして以下をカスタマイズできます：

Supported File Formats

サポートされているファイル形式

NectarView supports the following file formats:

NectarViewは以下のファイル形式をサポートしています：

  • Images: PNG, JPEG, GIF, BMP, TIFF, WebP
  • 画像：PNG、JPEG、GIF、BMP、TIFF、WebP

  • ドキュメント：PDF
