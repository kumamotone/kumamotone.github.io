NectarView - Privacy Policy

NectarView - プライバシーポリシー

Last Updated: September 30, 2025

最終更新日: 2025年9月30日

Overview

概要

NectarView is a local image viewer application for macOS that prioritizes user privacy. This privacy policy explains how we handle your information when you use our application.

NectarViewは、ユーザーのプライバシーを最優先に考えたmacOS用のローカル画像ビューアアプリケーションです。本プライバシーポリシーでは、アプリケーションの使用時における情報の取り扱いについて説明します。

Data Collection

データ収集について

NectarView does not collect, transmit, or store any personal information on external servers. All data processing occurs locally on your device.

NectarViewは、個人情報を収集、送信、または外部サーバーに保存することはありません。すべてのデータ処理はお客様のデバイス上でローカルに行われます。

Local Data Storage

ローカルデータストレージ

NectarView stores the following information locally on your device:

NectarViewは以下の情報をお客様のデバイス上にローカルに保存します：

  • Application Settings: Your preferences such as view mode, background color, and display options
  • アプリケーション設定：表示モード、背景色、表示オプションなどの設定
  • Bookmarks: File locations and page positions you have bookmarked
  • ブックマーク：ブックマークしたファイルの場所とページ位置
  • Recent Files: File paths of recently opened images or archives
  • 最近使用したファイル：最近開いた画像やアーカイブファイルのパス

This data is stored using standard macOS APIs and remains on your device at all times.

このデータは標準のmacOS APIを使用して保存され、常にお客様のデバイス上に保持されます。

File Access

ファイルアクセス

NectarView accesses files only when:

NectarViewは以下の場合にのみファイルにアクセスします：

  • You explicitly open files through the application interface
  • アプリケーションのインターフェースを通じて明示的にファイルを開く場合
  • You drag and drop files onto the application
  • アプリケーションにファイルをドラッグ&ドロップする場合
  • You use the "Open with NectarView" context menu option
  • 「NectarViewで開く」コンテキストメニューオプションを使用する場合

The application only reads files necessary for image display and does not modify, copy, or transmit your files.

アプリケーションは画像表示に必要なファイルのみを読み取り、ファイルの変更、コピー、送信は行いません。

Network Activity

ネットワーク活動

NectarView does not connect to the internet or any external servers. The application operates entirely offline and does not transmit any data over the network.

NectarViewはインターネットや外部サーバーに接続しません。アプリケーションは完全にオフラインで動作し、ネットワーク経由でデータを送信することはありません。

Third-Party Services

第三者サービス

NectarView does not integrate with any third-party services, analytics platforms, or advertising networks.

NectarViewは第三者サービス、分析プラットフォーム、広告ネットワークと統合されていません。

Data Security

データセキュリティ

Since all data remains on your device and no network transmission occurs, your data security depends on your device's security measures. We recommend keeping your macOS system updated and using appropriate security settings.

すべてのデータがお客様のデバイス上に保持され、ネットワーク送信が発生しないため、データのセキュリティはお客様のデバイスのセキュリティ対策に依存します。macOSシステムを最新の状態に保ち、適切なセキュリティ設定を使用することをお勧めします。

Changes to This Policy

本ポリシーの変更

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this page. Continued use of NectarView after any changes constitutes acceptance of the updated policy.

本プライバシーポリシーは随時更新される場合があります。変更があった場合は、このページの上部にある「最終更新日」に反映されます。変更後もNectarViewを継続して使用することは、更新されたポリシーに同意したものとみなされます。

Contact Information

お問い合わせ

If you have any questions about this privacy policy, please contact us through our GitHub repository.

本プライバシーポリシーについてご質問がございましたら、GitHubリポジトリを通じてお問い合わせください。
