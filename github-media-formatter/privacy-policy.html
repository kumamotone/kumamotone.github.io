Privacy Policy

GitHub Media Formatter

The GitHub Media Formatter extension does not collect, store, or transmit any user data.

This extension operates solely to format markdown media content in GitHub textareas and does not:

  • Collect personal information
  • Track browsing history
  • Use cookies
  • Send data to external servers
  • Implement analytics or tracking

This extension functions locally and has the single purpose of formatting media markdown on GitHub.
