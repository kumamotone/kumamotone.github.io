集中力を、取り戻す。

Focus Launcher は、あなたの注意力を取り戻すためのAndroidランチャーです。アプリを消さなくていい。意志力に頼らなくていい。ルールを一度決めれば、あとは静かに動き続ける。

なぜ作ったか

「シンプルなランチャーは、
シンプルすぎた。」

私はかつて、重度のスマホ中毒でした。気づけば一日中、SNSのタイムラインやショート動画を縦スワイプしていました。

転機になったのは、シンプルなランチャー、アプリブロックツール、使用統計ツールとの出会いでした。「使いすぎているアプリを見えにくくする」という発想が、自分には効きました。

ただ、使い続けるうちに限界を感じるようになりました。「自宅のWi-Fiに接続しているときだけXをブロックする」ような条件付きの制限には、ランチャー以外のツールが必要でした。使用状況を確認するには別のアプリが必要で、それらは複数の課金体系を持ち、広告も入っていました。

シンプルさにも落とし穴がありました。GoogleとMicrosoftのAuthenticatorアプリのように、アイコンをなくしてしまうと区別がつかなくなるアプリがあります。シンプルすぎることも、それはそれで不便でした。

自分が本当に欲しかったのは、一つのアプリでした。そんな自分のために作ったのが、Focus Launcherです。

Feature 01 · Block Rules

気になるアプリを、
少しだけ遠くに置く。

削除じゃない。WiFiや時間帯、集中モードを条件にアプリをブロックする。アプリはそこにある、ただ少し遠い。その距離が、衝動的に開く前の一呼吸になる。

Feature 02 · Usage Stats

自分でも気づいていなかった
パターンが見える。

「そんなに使っていないつもり」が一番危ない。使用状況は事実だけを静かに並べる。ゲーミフィケーションなし。ストリークなし。ただ、あなたの時間の使い方が、そこにある。

Feature 03 · Home Screen

アイコンを捨てたら、
開く前に考えるようになった。

グリッドなし、バッジなし。時刻と自分が選んだアプリの名前だけ。スクロールの先に何もない入口が、衝動的な操作を静かに止める。

Feature 04 · Journal

衝動を言葉にしたとき、
変化が始まる。

ブロックされたとき、なぜ開こうとしたかをメモできる。振り返ることで、自分の行動の引き金に気づき始める。
