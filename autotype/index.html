特徴

Features

リアルタイム書き起こし

Real-time Transcription

OpenAI の音声認識モデルでストリーミング書き起こし。話しながらテキストがリアルタイムで表示されます。

Streaming transcription powered by OpenAI. See your words appear as you speak.

2段階 AI 整形

Two-stage AI Refinement

Gemini による軽量リアルタイム整形（句読点・用語）+ Claude による深い整形（口語→書き言葉）をオンデマンドで。

Light real-time polish with Gemini (punctuation & terms) + deep refinement with Claude (spoken to written) on demand.

OS 全体で動作

Works System-wide

メニューバーに常駐し、どのアプリのテキストフィールドにも入力。Fn キーでいつでも起動。

Lives in the menu bar, types into any text field in any app. Activate anytime with the Fn key.

カスタム用語集

Custom Glossary

技術用語や固有名詞を登録すれば、正しい表記で自動変換。「タイプスクリプト」→「TypeScript」。

Register technical terms and proper nouns for accurate conversion. "タイプスクリプト" becomes "TypeScript".

Push-to-Talk 対応

Push-to-Talk

Fn 長押しで話し、離すと停止。短い発話に最適な操作モード。

Hold Fn to speak, release to stop. Ideal for quick dictations.

録音履歴

Session History

過去のセッションを音声付きで保存・再生。書き起こし結果のセグメントごとの確認も可能。

Save and replay past sessions with audio. Review transcription segment by segment.

セットアップ

Setup

必要なもの

Requirements

  1. macOS 14 Sonoma 以降
  2. macOS 14 Sonoma or later
  3. OpenAI API キー（音声認識用）
  4. OpenAI API key (for transcription)
  5. Gemini API キー（軽量整形用、任意）
  6. Gemini API key (for light refinement, optional)
  7. Anthropic API キー（深い整形用、任意）
  8. Anthropic API key (for deep refinement, optional)

インストール手順

Installation

  1. 最新リリースから AutoType.dmg をダウンロード
  2. Download AutoType.dmg from the latest release
  3. AutoType.app を Applications フォルダにドラッグ
  4. Drag AutoType.app to the Applications folder
  5. 初回起動時にアクセシビリティ権限とマイク権限を許可
  6. Grant accessibility and microphone permissions on first launch
  7. 設定画面で API キーとバックエンド URL を入力
  8. Enter your API keys and backend URL in Settings
